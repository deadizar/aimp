# Documentation Hub

This directory centralizes reusable process and session documentation for this repository.

## Main areas
- `src/docu/learned/` - reusable lessons learned and indexed knowledge.
- `src/docu/planLogs/` - immutable copies of agent plans by branch/date.
- `src/docu/diaryLogs/` - immutable copies of daily diary logs.
- `src/docu/continue/` - continuation prompts for the next session.
- `src/docu/daily/` - session summaries by date.
- `src/docu/ToDo/` - next-session actionable task lists.
- `src/docu/scripts/PyScripts/` - archived scripts and script documentation.
- `src/docu/skills/` - recurring-pattern tracking for future skill extraction.
