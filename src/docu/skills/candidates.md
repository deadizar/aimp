# Skills Scout Candidates

Track recurring workflow friction and repeated patterns to decide whether a reusable skill should be created.

## Format
`- [YYYY-MM-DD] <signal> | count=<n> | note=<short note>`

## Signals
- Repeatedly reading the same learned note in one session.
- Repeatedly forgetting a mandatory routine.
- User reminders for the same operational step.

If a candidate reaches 3 independent sessions, propose formalizing it into a reusable skill.
